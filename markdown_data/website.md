Website
=======

**Odoo Website** offers a user-friendly platform for creating and
managing your website. It includes various tools and features to help
you design, publish, and maintain web pages without needing advanced
technical skills. You can easily customize layouts, add multimedia
content, and integrate with other Odoo apps to expand your website\'s
functionality.

::: {.cards}
::: {.card target="website/web_design" large=""}
Web design

Design your website using building blocks and website themes.
:::

::: {.card target="website/pages" large=""}
Pages

Manage website pages, menus, and search engine optimization.
:::

::: {.card target="website/configuration" large=""}
Configuration

Configure domain names, address autocompletion, Google Search Console,
cookies bar, translations, multiple websites, form spam protection,
content delivery network (CDN).
:::

::: {.card target="website/reporting"}
Reporting

Monitor your website\'s traffic with website analytics and set up link
trackers.
:::

::: {.card target="website/mail_groups"}
Mail groups

Configure mail groups to allow website visitors to participate in public
discussions via email.
:::
:::

::: {.note}
::: {.title}
Note
:::

Odoo offers a
`free custom domain name <domain-name/register>`{.interpreted-text
role="ref"} to all Odoo Online databases for one year. Visitors can then
access your website with an address such as
[www.example.com]{.title-ref} rather than the default
[example.odoo.com]{.title-ref}.
:::
