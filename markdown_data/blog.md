Blog
====

**Odoo Blog** allows you to create and manage blog posts on your website
to engage your audience and build a community.

::: {.tip}
::: {.title}
Tip
:::

If the Blog module is not yet installed, click `+New`{.interpreted-text
role="guilabel"} on the website builder, select
`Blog Post`{.interpreted-text role="guilabel"}, and click
`Install`{.interpreted-text role="guilabel"}.
:::

Creating a blog
---------------

To create or edit a blog, go to
`Website --> Configuration --> Blogs: Blogs`{.interpreted-text
role="menuselection"}. Click `New`{.interpreted-text role="guilabel"},
and enter the `Blog Name`{.interpreted-text role="guilabel"} and the
`Blog Subtitle`{.interpreted-text role="guilabel"}.

The `Blog`{.interpreted-text role="guilabel"} menu item is added to your
website\'s menu the first time you create a blog and gathers all your
blogs.

Adding a blog post
------------------

Go to your website, click `+New`{.interpreted-text role="guilabel"} in
the top-right corner, and select `Blog Post`{.interpreted-text
role="guilabel"}. In the pop-up, **select the blog** where the post
should appear, write the post\'s `Title`{.interpreted-text
role="guilabel"}, and `Save`{.interpreted-text role="guilabel"}. You can
then write the post\'s content and customize the page using the website
builder.

::: {.tip}
::: {.title}
Tip
:::

\- Illustrate your articles with copyright-free images from `Unsplash
</applications/general/integrations/unsplash>`{.interpreted-text
role="doc"}. - Type [/]{.title-ref} in the text editor to format and add
elements to your text.
:::

::: {.important}
::: {.title}
Important
:::

Don\'t forget to toggle the `Unpublished`{.interpreted-text
role="guilabel"} switch in the top-right corner to publish your post.
:::

Using tags
----------

Tags let visitors filter all posts sharing the same tag. By default,
they are displayed at the bottom of posts, but can also be displayed on
the blog\'s main page. To do so, click
`Edit --> Customize`{.interpreted-text role="menuselection"} and enable
the `Sidebar`{.interpreted-text role="guilabel"}. By default, the
sidebar\'s `Tags List`{.interpreted-text role="guilabel"} is enabled.

To create a tag, go to
`Website --> Configuration --> Blogs: Tags`{.interpreted-text
role="menuselection"} and click `New`{.interpreted-text
role="guilabel"}. Fill in the:

-   `Name`{.interpreted-text role="guilabel"}
-   `Category`{.interpreted-text role="guilabel"}: tag categories let
    you group tags displayed on the sidebar by theme.
-   `Used in`{.interpreted-text role="guilabel"}: to apply the tag to
    existing blog posts, click `Add a line`{.interpreted-text
    role="guilabel"}, select the posts, and click
    `Select`{.interpreted-text role="guilabel"}.

You can add and create tags directly from posts by clicking
`Edit --> Customize`{.interpreted-text role="menuselection"} and select
the post\'s cover. Under `Tags`{.interpreted-text role="guilabel"},
click `Choose a record...`{.interpreted-text role="guilabel"}, and
select or create a tag.



::: {.note}
::: {.title}
Note
:::

To manage tag categories, go to
`Website --> Configuration --> Blogs: Tag
Categories`{.interpreted-text role="menuselection"}.
:::

Customizing blog homepages
--------------------------

Customize the content of blog homepages by opening a blog homepage and
clicking `Edit
--> Customize`{.interpreted-text role="menuselection"}.

::: {.note}
::: {.title}
Note
:::

Settings apply to **all** blogs homepages.
:::

`Top Banner`{.interpreted-text role="guilabel"}:
`Name/Latest Post`{.interpreted-text role="guilabel"} displays the title
of the latest post on the top banner, while
`Drop Zone for Building Blocks`{.interpreted-text role="guilabel"}
removes the top banner and lets you use any building block instead.

`Layout`{.interpreted-text role="guilabel"}: organizes posts as a
`Grid`{.interpreted-text role="guilabel"} or `List`{.interpreted-text
role="guilabel"}.

-   `Cards`{.interpreted-text role="guilabel"}: adds a *card* effect.
-   `Increase Readability`{.interpreted-text role="guilabel"}: improves
    the text\'s readability.

`Sidebar`{.interpreted-text role="guilabel"}: displays a sidebar
containing an `About us`{.interpreted-text role="guilabel"} section.

-   `Archives`{.interpreted-text role="guilabel"}: allows visitors to
    select a month and filter all posts created during that month.
-   `Follow Us`{.interpreted-text role="guilabel"}: displays links to
    your social media networks. They can be configured using the Social
    Media building block somewhere on your website.
-   `Tags List`{.interpreted-text role="guilabel"}: displays all tags
    related to a blog. Visitors can select a tag to filter all related
    posts.

`Posts List`{.interpreted-text role="guilabel"}:
`Cover`{.interpreted-text role="guilabel"} displays the posts\' images,
and `No Cover`{.interpreted-text role="guilabel"} hides them.

-   `Author`{.interpreted-text role="guilabel"}: displays the posts\'
    authors.
-   `Comments/Views Stats`{.interpreted-text role="guilabel"}: displays
    the posts\' number of comments and views.
-   `Teaser & Tags`{.interpreted-text role="guilabel"}: displays the
    posts\' first sentences and tags.

Customizing blog posts
----------------------

Customize posts by opening a blog post and clicking
`Edit --> Customize`{.interpreted-text role="menuselection"}.

::: {.note}
::: {.title}
Note
:::

Settings apply to **all** posts.
:::

`Layout`{.interpreted-text role="guilabel"}:
`Title Inside Cover`{.interpreted-text role="guilabel"} displays the
title inside the cover image, and `Title above Cover`{.interpreted-text
role="guilabel"} displays it above.

-   `Increase Readability`{.interpreted-text role="guilabel"}: increases
    the text\'s readability.

`Sidebar`{.interpreted-text role="guilabel"}: displays the
`Sidebar`{.interpreted-text role="guilabel"} and additional options:

-   `Archive`{.interpreted-text role="guilabel"}: allows visitors to
    select a month and filter all posts created during that month.
-   `Author`{.interpreted-text role="guilabel"}: displays the post\'s
    author and creation date.
-   `Blog List`{.interpreted-text role="guilabel"}: displays links to
    all your blogs.
-   `Share Links`{.interpreted-text role="guilabel"}: displays share
    buttons to several social networks.
-   `Tags`{.interpreted-text role="guilabel"}: displays the post\'s
    tags.

`Breadcrumb`{.interpreted-text role="guilabel"}: displays the path to
the post.

`Bottom`{.interpreted-text role="guilabel"}:
`Next Article`{.interpreted-text role="guilabel"} displays the next post
at the bottom, and `Comments`{.interpreted-text role="guilabel"} enable
visitors to comment on the post.

`Select To Tweet`{.interpreted-text role="guilabel"}: visitors are
offered to tweet the text they select.

::: {.tip}
::: {.title}
Tip
:::

Use `Plausible <analytics/plausible>`{.interpreted-text role="ref"} to
keep track of the traffic on your blog.
:::
